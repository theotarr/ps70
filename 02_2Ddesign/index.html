Week 2 — 2D Design

Goals

  • Learn sketch constraints and dimensions
  • Practice extruding and shelling to create hollow parts (mugs)
  • Explore patterning and revolve for rounded features
  • Create a simple fan blade and hub, test boolean joins

Fan build — from sketch to frame and blades

  1. Sketch the outer frame
    New sketch on the Top plane. Draw a 700 × 700 mm square centered on the origin. Add four corner fillets: inner profile R50 mm and outer profile R75 mm to get a rounded-square. Constrain and fully dimension.
  2. Make the frame body
    Finish Sketch → Extrude the ring (select outer minus inner loop) to ~20–25 mm. Add model Fillets (3–5 mm) to exterior and interior edges to soften.
  3. Sketch one blade
    New sketch on the same plane. Draw a teardrop blade using tangent arcs or a spline; dimension chord and width. Add a center construction circle for the hub axis if needed.
  4. Extrude and circular pattern
    Extrude blade to 8–10 mm. Create a cylindrical hub, then use Circular Pattern (Features) around the hub axis with 3 instances. Combine → Join to merge or keep separate for edits.

Tips: Keep blade curves tangent and symmetric. Use Shell on the frame if you want a hollow section.

Mug build — revolve, shell, and handle

  1. Profile for revolve
    Sketch half cross‑section on the Front plane with a vertical axis centerline. Use Offset to define wall thickness. Add blended bottom radii using two arcs. Fully constrain: height, rim OD, thickness, and radii.
  2. Revolve and shell
    Revolve the profile around the centerline to create the solid. Apply Shell from the top face to target 3–4 mm wall thickness.
  3. Handle sketch and extrude
    On a side plane, sketch a tapered handle strip with smooth arcs; dimension width and clearance. Extrude as a new body so the ends intersect the mug.
  4. Join and refine
    Combine → Join to attach handle. Add generous Fillets at the junction (3–6 mm) and a small rim fillet for comfort.
  5. Optional tweaks
    Apply Draft for a slight taper and add Appearance for renderings.

Printability notes: keep minimum wall ≥ 2.4–3.0 mm for a 0.4 mm nozzle; avoid unsupported overhangs > 45° around the handle.
